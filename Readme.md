## Как запустить:
Задание выполнено на [Go](https://golang.org)
1) [Скачать go](https://golang.org/dl/)
2) Перейти в корень проекта
3)
```
go mod download
go build 
./dnsServer
```

Для выполнения спросит root (запускается на порту 53, как и все dns сервера)
